DataCite Metadata Schema Repository

Current version

The preferred schema as of 9th May 2011 is DataCite Metadata Kernel 2.1.

Deprecation notification

DataCite Metadata Kernel Version 2.0 will be retired on 1st Aug 2011. After this date MDS will not accept documents based on this version.

Versioning

Each available version of a schema is a subdirectory containing a set of files. Those files constitute single metadata version. There are all the required xsd files, a sample xml file and a pdf documentation. The files don't reference any other files outside of their version subdirectory.

For our Metadata store all versions hosted on this site are valid although only one version is considered preferred at any given moment of time. Please note that each version has a different XML namespace.

Each incoming to Metadata store XML document should indicate schema location in the root element.
